基于 vite + react + ts 的 spa 脚手架

## 特性

- antd
  - 打包后默认为暗色主题（设置的亮色主题）
- less
- redux
- swr
- ahooks
- tailwindcss
  - 现存问题：tailwindcss 会覆盖 antd 的基础样式
- eslint
  - 现存问题：编辑器没有显示错误信息
